


### Node Js Setup

Linux commands to setup node:
```
sudo apt install nodejs
sudo apt install npm

```
_Node Js version should be above 14.6.0_
```
node -v
```

### Database Setup
```
npm i drizzle-orm @neondatabase/serverless dotenv
npm i -D drizzle-kit tsx
npm i dotenv
```
